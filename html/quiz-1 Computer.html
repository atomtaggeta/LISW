Quiz

Fill in the Blanks

  1. The central processing unit (CPU) is often referred to as the __________ of the computer. (Answer: brain)
  2. The __________ is responsible for managing memory and executing instructions. (Answer: CPU)
  3. In a computer, the __________ stores data temporarily for quick access by the CPU. (Answer: RAM)
  4. The __________ bus is used to transfer data between the CPU and memory. (Answer: data)
  5. The __________ is a non-volatile storage device used to store firmware. (Answer: ROM)

Multiple Choice Questions

  1. Which of the following is NOT a component of the CPU?
    • a) ALU
    • b) Control Unit
    • c) RAM
    • d) Registers
    (Answer: c) RAM)
  2. What does RAM stand for?
    • a) Read Access Memory
    • b) Random Access Memory
    • c) Readable Access Memory
    • d) Randomized Access Memory
    (Answer: b) Random Access Memory)
  3. Which of the following is a type of secondary storage?
    • a) CPU
    • b) Cache
    • c) Hard Disk Drive
    • d) Register
    (Answer: c) Hard Disk Drive)
  4. The primary function of the ALU (Arithmetic Logic Unit) is to:
    • a) Store data
    • b) Perform arithmetic and logical operations
    • c) Control input/output devices
    • d) Manage memory
    (Answer: b) Perform arithmetic and logical operations)

True or False Questions

  1. The control unit directs the operation of the processor and its interaction with other components. (True/False) (Answer: True)
  2. Cache memory is slower than RAM. (True/False) (Answer: False)
  3. A motherboard connects all the components of a computer. (True/False) (Answer: True)
  4. Flash memory is a type of volatile memory. (True/False) (Answer: False)
