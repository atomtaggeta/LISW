True or False

1. The width of the data bus is the number of lines of the data bus, which determines the number of bits that can be transmitted at the same time. True
2. The address bus determines the addressing capability of the CPU, and if you use the address bus for N bits, the accessible space is 2n. True
3. Software is divided into system software and application software. True
4. The operating system is software that can only be read and cannot be written and stored in the ROM. False
5. Software design should avoid modularity. False
6. UML, structure chart and class diagrams may be used during software design. True
7. Information shielding between modules in the software design process means that each other's information cannot be accessed arbitrarily. True
8. In the process of software implementation, try to avoid using pseudocode and write in a specific language. False
9. Signed integers typically use the first 0 to mean negative numbers. False
10. For positive numbers, we use the same bit patterns for a number no matter which scheme we use: sign-and-magnitude, one’s complement, and two’s complement. True
11. If we two’s complement a number twice, we get the original number. True
12. Java has fast speed and can solve problems with the help of pointers. False
13. Waterfall model is useful for dealing with problems that change frequently and have no concrete requirements. False
14. The system software includes device drivers. True
15. Word process software and spreadsheet software both belong to system software. False
