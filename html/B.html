Blockchain
A decentralized digital ledger that records transactions across many computers.
Big Data
Extremely large data sets that may be analyzed computationally to reveal patterns.
Byte
A unit of digital information that consists of 8 bits.
Buffer
A temporary storage area typically used to hold data while it is being moved from one place to another.
Bug
An error or flaw in software or hardware that causes it to produce an incorrect or unexpected result.
Bandwidth
The maximum rate of data transfer across a network.
Binary
A numbering system that uses only two digits, 0 and 1, to represent data.
Boolean
A data type that can hold one of two possible values: true or false.
Backpropagation
An algorithm used for training artificial neural networks through a method of optimizing weights.
Benchmark
A standard measure used to compare the performance of various components or systems.
Browser
A software application used to access and navigate the internet.
Bot
A software application that runs automated tasks over the internet.
Bytecode
An intermediate representation of a program, typically used in the context of compiled languages.
Bootstrap
A front-end framework for developing responsive websites and applications.
Batch Processing
A method of processing data where multiple jobs are processed at the same time without user interaction.
Bidirectional
Refers to communication paths that can send and receive data in both directions.
Bioinformatics
An interdisciplinary field that uses software tools to understand biological data.
Business Intelligence
Technologies and practices for collecting, analyzing, and presenting business data.
Brute Force
A method of solving problems through exhaustive search, trying all possible solutions.
Broadcast
A method of sending data to multiple recipients simultaneously over a network.
