GUI (Graphical User Interface)
A user interface that includes graphical elements, such as windows, icons, and buttons, allowing users to interact with electronic devices easily.
Git
A distributed version control system used for tracking changes in source code during software development.
GDB (GNU Debugger)
A powerful debugging tool for programs written in C, C++, and other languages, allowing programmers to analyze and control program execution.
Grid Computing
A distributed computing model that harnesses the power of multiple computers to work together on complex tasks.
Graph
A data structure that consists of nodes connected by edges, commonly used for representing networks.
Garbage Collection
Automatic memory management that frees up memory occupied by objects that are no longer in use.
Gateway
A network point that acts as an access point to another network, often translating protocols between different networks.
Generator
A function or tool that produces a sequence of values or objects, often used in programming to manage memory efficiently.
Grading System
A systematic method for evaluating and scoring student performance in an educational context, often implemented in educational software.
Gesture Recognition
A technology that allows a user to interact with devices through gestures, often using camera inputs.
Google Cloud
A suite of cloud computing services offered by Google that runs on the same infrastructure that Google uses internally for its end-user products.
Gamification
The use of game-design elements in non-game contexts, such as education and business, to engage users and enhance user experiences.
Globalization
A development process that connects different cultures and economies through technology, particularly affecting software localization and design.
Groupware
Software designed to help people work together, such as online collaboration tools and communication platforms.
