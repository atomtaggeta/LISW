Big Data

LEARING SUGGESTIONS

1. Big data primarily refers to data sets that are too large or complex to be dealt with by traditional data-processing application software.

2. Big data analysis include capturing data, data storage, data analysis, search, sharing, transfer, visualization, querying, updating, information privacy, and data source.

3. Big data was originally associated with three key concepts: volume, variety, and velocity.
