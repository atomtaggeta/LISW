True or False

1. The principle of stored program control is the core of von Neumann's thought. True
2. The GPU enables the graphics card to reduce the dependence on the CPU and undertake part of the original CPU work, especially when performing 3D graphics processing. True
3. A microcomputer is based on the CPU, and other components are all "connected" to the system bus connected to the CPU. True
4. The coprocessor is also a kind of chip, which is used to reduce the specific processing tasks of the system microprocessor. True
5. The main function of the memory is to store currently used programs and data. True
6. Cache is external storage. False
7. The main technical indicators of the printer are printing quality, speed, and noise. True
8. One factor that determines the computing speed of a computer is the number of instructions that can be executed per second. True
9. Read-only memory is a memory specially used to read content, but the system must write content for it before powering on each time. False
